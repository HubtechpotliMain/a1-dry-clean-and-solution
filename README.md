## üöÄ Getting Started

### Prerequisites
- Node.js 18+
- npm, yarn, or pnpm

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/hubtechpotli/ai_dry_clean.git
   cd ai_dry_clean
   ```

2. **Install dependencies**
   ```bash
   cd Frontend
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)
